# BALF细胞标注系统 - 深度优化与数据管理系统构建任务

## 🔴 项目背景：紧急修复与升级

这是一个**被恶意破坏后部分还原**的医学图像细胞标注系统。当前系统虽能运行，但存在严重的功能缺陷，特别是在**数据管理**和**标注效率**方面。我们需要在保留现有核心功能的基础上，进行全面优化和新功能构建。

### 当前系统现状
- **后端**: FastAPI单体应用 (~2600行)，基础API可用
- **前端**: 原生JS，标注核心功能可用但被删减过
- **AI模块**: SAM3、Cellpose、BiomedCLIP已集成，功能正常
- **数据存储**: 纯文件系统，YOLO格式文本文件

### 必须保留的核心功能
- SAM3智能分割（框选/文本/点提示）
- Cellpose细胞自动分割
- BiomedCLIP少样本分类
- YOLO格式导入导出
- 基础的多边形/矩形标注

## 🎯 核心任务：两大系统建设

# 第一部分：原有标注系统深度优化

## 1. 标注效率极致优化

### 1.1 快捷键系统全面重构
当前系统缺乏快捷键，严重影响标注效率。

**必须实现的快捷键**:
```
【工具切换】
V - 选择工具（Select）
R - 矩形工具（Rectangle）
P - 多边形工具（Polygon）
C - 圆形工具（Circle）
O - 椭圆工具（Oval）
B - 画笔工具（Brush）
E - 橡皮擦（Eraser）
S - SAM智能分割
M - 测量工具
H - 手掌拖拽（按住空格临时切换）

【编辑操作】
Ctrl+Z - 撤销
Ctrl+Y / Ctrl+Shift+Z - 重做
Ctrl+C - 复制选中标注
Ctrl+V - 粘贴标注
Ctrl+D - 复制并偏移
Delete - 删除选中标注
Ctrl+A - 全选
Esc - 取消当前操作/取消选择
Enter - 完成多边形绘制

【视图控制】
Ctrl++ - 放大
Ctrl+- - 缩小
Ctrl+0 - 适应屏幕
Ctrl+1 - 实际像素
Ctrl+滚轮 - 缩放
滚轮 - 垂直滚动
Shift+滚轮 - 水平滚动
按住空格+拖拽 - 平移

【类别切换】
数字键 1-9 - 快速切换到对应类别
Shift+数字键 - 为选中标注设置类别

【图像导航】
← → 或 A D - 上一张/下一张图像
Ctrl+← → - 跳转到未标注图像
Ctrl+Shift+← → - 跳转到有问题的图像
F - 标记当前图像为完成/待审核

【AI辅助】
Tab - 自动分割当前选中区域
Shift+Tab - AI推荐下一个标注位置
Ctrl+Enter - 应用AI预标注到当前图像
```

**进阶功能**:
- 快捷键自定义配置（持久化到用户配置）
- 快捷键冲突检测
- 快捷键提示面板（按?键显示）
- Vim模式支持（可选高级功能）

### 1.2 标注工具增强

#### 多边形工具优化
- **绘制优化**:
  - 支持双击自动闭合
  - 支持回退上一点（Backspace）
  - 实时显示边长和角度
  - 吸附到图像边缘（磁吸距离可配置）
  - 绘制时显示面积预览

- **编辑增强**:
  - 顶点拖拽时显示对齐线（类似Photoshop）
  - 支持在边上添加新顶点（双击边）
  - 支持删除顶点（右键点击/选中按Delete）
  - 支持顶点合并（距离近的顶点自动合并）
  - 支持多边形简化（减少顶点数，保持形状）
  - 支持多边形平滑（曲线化）

#### 矩形工具增强
- 绘制时显示尺寸（像素/微米）
- 支持旋转（按住Shift画正方形，绘制后可旋转）
- 支持圆角矩形
- 拖拽调整大小时保持比例（按住Shift）

#### 新增画笔工具
- 自由绘制mask
- 可调节笔刷大小（[]键快速调整）
- 支持压感（如果硬件支持）
- 和Cellpose/SAM结合：画笔粗略涂抹，AI精确分割

#### 新增橡皮擦工具
- 擦除mask区域
- 可调节橡皮大小
- 支持矩形/圆形擦除模式

### 1.3 智能辅助功能

#### 智能吸附
- **顶点吸附**: 新顶点靠近已有顶点时自动吸附
- **边缘吸附**: 新边靠近已有边时自动对齐
- **网格吸附**: 可配置网格大小（10px/50px/100px）
- **中心吸附**: 自动吸附到图像中心线

#### 智能填充
- 基于SAM的一键填充：点击细胞内部自动分割边界
- 智能扩展：选中区域按方向键扩展像素
- 孔洞填充：自动检测并填充mask内部孔洞

#### 批量操作
- 框选多个标注进行批量移动/删除
- 批量修改类别
- 批量复制到相邻图像（针对连续切片）
- 批量AI预标注（选择图像范围，后台处理）

### 1.4 撤销重做系统重构

当前系统的undo功能不够完善，需要重构：

- **深度栈**: 支持100+步撤销
- **操作类型记录**: 
  - 标注创建/删除/修改
  - 顶点移动
  - 类别变更
  - 属性修改
- **分支支持**: 撤销后新操作不直接清空redo栈，而是创建分支（可选）
- **持久化**: 标注会话的undo栈自动保存，刷新页面可恢复
- **可视化**: Ctrl+Z长按显示操作历史列表

### 1.5 图像导航优化

#### 缩略图导航
- 右侧显示缩略图列表
- 缩略图上显示标注进度指示
- 缩略图支持筛选（只看未标注/有问题）
- 缩略图支持多选进行批量操作

#### 智能跳转
- 按类别筛选图像（只看包含某类别的）
- 按标注数量筛选（找标注少/多的）
- 按文件名搜索跳转
- 标记问题图像，快速跳转到待修正的

#### 进度保存与恢复
- 自动保存当前标注状态（每30秒）
- 关闭页面后重新打开，恢复到上次位置
- 多标签页同步（防止同时编辑同一图像）

## 3. 标注数据管理增强

### 3.1 标注列表面板

右侧新增标注列表面板：
- 按类别分组显示所有标注
- 点击列表项选中画布对应标注
- 支持拖拽排序
- 支持搜索筛选
- 支持批量删除/修改类别

### 3.2 标注属性系统

为标注添加属性字段：
```
细胞类别: 中性粒细胞
  ├─ 成熟度: 成熟/未成熟
  ├─ 状态: 正常/凋亡/坏死
  ├─ 分裂相: 是/否
  └─ 备注: 文本

标注置信度: 人工标注(1.0) / AI辅助(0.85)
```

- 属性定义在项目级别配置
- 属性值可筛选统计
- 属性导出到YOLO自定义字段

### 3.3 标注冲突检测

- **重叠检测**: 相同类别标注重叠时警告
- **边界检测**: 标注超出图像边界时警告
- **大小异常**: 过大/过小标注警告
- **孤立点**: 单个像素标注自动清理

## 5. 任务分配与协作系统

### 5.1 用户角色系统

```
管理员(Admin)
  └─ 创建项目、管理用户、查看所有数据

项目经理(Project Manager)
  └─ 管理分配任务、查看统计、审核

  └─ 只读访问
```

### 5.2 任务分配

- 按图像数量平均分配

- 可视化分配界面
- 拖拽分配图像给标注员
- 批量分配/取消分配
- 设置截止日期

#### 任务队列
- 标注员视角：待标注队列、进行中、已完成

- 难度标记：简单/中等/困难

### 5.3 审核工作流

#### 审核流程配置
```
单级审核: 标注 → 审核 → 完成
多级审核: 标注 → 初审 → 复审 → 完成
抽查审核: 标注 → 抽查 → 完成
```

#### 审核界面
- 并排的标注视图和原图
- 问题标记工具
- 评论留言
- 快速操作（通过/需修正/拒绝）

#### 审核统计
