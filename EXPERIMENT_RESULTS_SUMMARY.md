## 1. 分类：BALF data2（主数据集）

### 1.1 数据集
- **名称**：`cell_datasets/data2_organized/`
- **样本量**：训练 5,315 个细胞实例，验证 1,316 个
- **类别**：Eos 4.4% / Neu 9.7% / Lym 54.9% / Mac 31.1%
- **来源**：BALF 样本，Cellpose 分割 + 细胞学医师人工标注

### 1.2 评估协议
- **Support**：每类 N=10（共 40）

## 2. 分割：WBC-Seg 数据集（金标准，标签已修正）

### 2.1 数据集
- **名称**：WBC Seg（YOLO-seg 格式）
- **路径**：`cell_datasets/WBC Seg/yolo_seg_dataset/`
- **规模**：验证集 152 张图（3120×4160）；共 **22,683 个 GT 多边形**
- **密度**：每张图约 **149 个细胞实例**（高密度血涂片视野）
- **GT 制作**：YOLO 多边形标注，所有坐标归一化在 [0, 1]（修正后）

### 2.2 方法
- **CellposeSAM**：Cellpose 4.1.1 `cpsam` 预训练权重，GPU 推理，diameter=30, cellprob_threshold=-2.0

### 2.3 验证集上的实例分割结果 @ IoU≥0.5

| 指标 | 值 |
|------|----|
| 总预测细胞数 | **21,038** |
| 总 GT 细胞数 | **22,683** |
| 匹配成功数 (TP @ IoU≥0.5) | **19,398** |
| **Overall Precision** | **0.9220** |
| **Overall Recall** | **0.8552** |
| **Overall F1** | **0.8874** |

**意义**：在完全独立的外部数据集（WBC Seg, 22k+ GT）上，CellposeSAM 的实例级 F1 达到 **0.887**，验证了本文分割前端的工业级可用性，消除了 BALF 数据集无独立分割 GT 的担忧。

### 2.4 BALF data2 分割（PAMSR 优化）

## 3. 跨数据集泛化：MultiCenter

### 3.1 数据集特点
- 规模：训练 1897 张 / 5242 细胞；验证 475 张 / 1349 细胞
- 类别分布（val）：Neu 86%, Mac 9.6%, Lym 3.8%, **Eos 仅 0.4%（5 个样本）**
- 图像尺寸：853×640 JPG（vs data2 的 2048×1536 PNG）

### 3.2 跨中心分类结果（10-shot, Nested CV）

| 方法 | data2 mF1 | MultiCenter mF1 | 说明 |
|------|-----------|-----------------|------|
| NCM (BiomedCLIP) | 0.6557 | **0.3798** | MC 上最佳 |
| kNN k=5 | 0.6582 | 0.3300 | — |
| MB-kNN | 0.7252 | 0.3190 | 类不平衡导致 ATD 失效 |
| **AFP-OD P3c** | **0.7563** | **0.3190** | 在 data2 上稳定最优 |

**结论**：MultiCenter 的极端类不平衡（Eos 仅 5 例）削弱了 AFP-OD 的优势；NCM 反而在此场景下最优，提示未来工作应考虑极端不平衡场景的适配。

## 6. 记录于本文档的阴性/失败结果

| 方法 | mF1 | 说明 |
